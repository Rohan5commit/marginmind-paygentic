# Demo Script

## 2–3 Minute Flow

### 1. Problem
“Small teams waste money every month on duplicate SaaS, rising infra spend, and renewals nobody catches in time. MarginMind is an autonomous agent built to find that waste and turn it into action.”

### 2. Detection
“Here I load a realistic startup spend file. In a few seconds the Intake and Detection Agents normalize the transactions, identify recurring charges, and surface the most valuable savings opportunities.”

### 3. Decision
“Now the Decision Agent has classified what to do: cancel the duplicate analytics tool, downgrade the underused video subscription, request a refund on a recent annual renewal, and negotiate a high-value monitoring contract.”

### 4. Locus Action
“This is the key part: MarginMind does not stop at analysis. The Action Agent maps each opportunity into a Locus-native execution path. It shows wallet-funded research, policy-aware approvals, and a Build with Locus deployment path for an always-on renewal monitor.”

### 5. Task Escalation
“For actions that need human finesse, like a vendor negotiation email, the Task Escalation Agent creates a Locus Tasks proposal with category, timeline, price tier, and expected ROI. It only escalates when the task cost is worth the savings.”

### 6. Savings Result
“At the top of the dashboard we can see the monthly waste estimate, annualized savings, and the highest-priority next move. The Strategy Agent then generates a concise founder-ready summary using NVIDIA NIM.”

### 7. Closing Impact Statement
“MarginMind is an agent that literally helps make you money by reducing wasteful spend. It uses Locus as the execution and payment layer, not just as a logo. That makes it a strong fit for Paygentic Week 2.”
